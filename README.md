# Basic Dashboard

A simple dashboard project developed using **TypeScript** and **ReactJS**. This dashboard displays some fake data for demonstration purposes, including employees, projects, and hotel data.

## Features

- Login page (enter **any email and password** to sign in)
- Dashboard displaying widgets with key metrics
- Employees, Projects, and Hotel data tables
- Fully responsive layout
- Built with React, TypeScript, and Vite

## How to Use

1. Visit the dashboard here: [Basic Dashboard](https://jeeson12.github.io/basic_dashboard/)
2. Enter **any email and password** on the login page.
3. Navigate through the dashboard sections using the sidebar.

## Tech Stack

- **ReactJS** for the frontend
- **TypeScript** for type safety
- **Vite** for project bundling
- **Tailwind CSS** for styling


---

This project is for learning and demo purposes. All data is fake and generated locally.
